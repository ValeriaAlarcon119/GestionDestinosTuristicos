# HotelBediaX - Frontend

Este es el cliente web para la plataforma de gestión de destinos turísticos **HotelBediaX**.

## 🚀 Tecnologías
- **React 19** + **TypeScript**
- **Vite 7**
- **React Router** para navegación
- **Axios** para consumo de API REST
- **Vanilla CSS** para estilos de alta calidad

## 📦 Gestión
Este módulo gestiona la interfaz de usuario para el mantenimiento de destinos:
- Visualización de datos en tablas reactivas.
- Búsqueda y filtrado dinámico.
- Formularios modales para creación y edición.
- Paginación del lado del servidor.

## 🏃 Cómo correrlo
1. Instalar dependencias: `npm install`
2. Correr en modo dev: `npm run dev`
3. Asegúrate de que el **Backend** esté corriendo en el puerto `5210` para que la API responda.
